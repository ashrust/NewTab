Overview

NewTab is simple new tab page with beautiful background images, simple search and no ads or trackers.

Every night, the app collects links to the top images from Reddit via Imgur. The default search engine is DuckDuckGo but you can hit tab to switch to Google.

Made by Ash Rust.

How to Install

Safari: Go to Safari's Preferences. Under the "General" tab (top left), set your homepage to: https://newtab.click/. Then set "New tabs open with" to "Homepage" and also set "New windows open with" to "Homepage".

Chrome: The chrome extension is currently awaiting approval.

Edge: Here's how to add Chrome extensions to Edge (scroll down).
